О странице

Здесь есть семантические теги, список, изображения с описательными alt и внешняя ссылка. Структура заголовков соблюдена: h1 → h2.

Список целей проекта

  1. Сделать корректную HTML-структуру (doctype, head, body, title, main).
  2. Добавить семантические элементы и логичную структуру заголовков.
  3. Проверить страницу через валидатор и инструмент доступности.

Галерея изображений

Все изображения имеют описательные alt-тексты.

Полезные ресурсы

  • Проверка валидности HTML: W3C Markup Validation Service
  • Проверка доступности: WAVE Web Accessibility Evaluation Tool
